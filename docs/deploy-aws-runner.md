## ☁ Using AWS App Runner

If using AWS App Runner, set the build settings as below:

- Build command:

  ```bash
  npm install && npm run build
  ```

- Start command:

  ```bash
  npm run start:prod
  ```

- Port:

  ```bash
  3000
  ```
